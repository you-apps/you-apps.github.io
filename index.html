Download and Verification

The only official sources for downloading You Appsare GitHub releases, F-Droidand IzzyOnDroid. The download links can be found in the GitHub repository of each app. Any other sources are unofficial and potentially malicious!

If you're downloading via GitHub releasesor IzzyOnDroid, please make sure to verify the checksums of the signing certificates.

Contribute now!

All contributions are very welcome!

  • Feel free to join the Matrix Room for discussions about the app.
  • Bug reports and feature requests can be submitted to our repository (please make sure to fill out all the requested information properly!).
  • If you are a developer and wish to contribute to the app, please forkthe project and submit apull request.
